Error: error:pget
Provider Missing ${symbol_dollar}get

Provider '{0}' must define ${symbol_dollar}get factory method.

Description

This error occurs when attempting to register a provider that does not have a ${symbol_dollar}get method. For example:

function BadProvider() {} // No ${symbol_dollar}get method!
angular.module("myApp", [])
  .provider(&${symbol_pound}39;bad&${symbol_pound}39;, BadProvider);  // this throws the error

To fix the error, fill in the ${symbol_dollar}get method on the provider like so:

function GoodProvider() {
  this.${symbol_dollar}get = angular.noop;
}
angular.module("myApp", [])
  .provider(&${symbol_pound}39;good&${symbol_pound}39;, GoodProvider);

For more information, refer to the ${symbol_dollar}provide.provider api doc.

Directive components in ngRoute

Name Description
ngView

Overview

ngView is a directive that complements the ${symbol_dollar}route service by including the rendered template of the current route into the main layout (index.html) file. Every time the current route changes, the included view changes with it according to the configuration of the ${symbol_dollar}route service.

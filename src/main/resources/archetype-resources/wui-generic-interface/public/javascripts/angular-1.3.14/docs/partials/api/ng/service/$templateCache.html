${symbol_dollar}templateCache

  1. - service in module ng

The first time a template is used, it is loaded in the template cache for quick retrieval. You can load templates directly into the cache in a script tag, or by consuming the ${symbol_dollar}templateCache service directly.

Adding via the script tag:

<script type="text/ng-template" id="templateId.html">
  <p>This is the content of the template</p>
</script>

Note: the script tag containing the template does not need to be included in the head of the document, but it must be a descendent of the ${symbol_dollar}rootElement (IE, element with ng-app attribute), otherwise the template will be ignored.

Adding via the ${symbol_dollar}templateCache service:

var myApp = angular.module(&${symbol_pound}39;myApp&${symbol_pound}39;, []);
myApp.run(function(${symbol_dollar}templateCache) {
  ${symbol_dollar}templateCache.put(&${symbol_pound}39;templateId.html&${symbol_pound}39;, &${symbol_pound}39;This is the content of the template&${symbol_pound}39;);
});

To retrieve the template later, simply use it in your HTML:

<div ng-include=" &${symbol_pound}39;templateId.html&${symbol_pound}39; "></div>

or get it via Javascript:

${symbol_dollar}templateCache.get(&${symbol_pound}39;templateId.html&${symbol_pound}39;)

Error: error:norslvr
No resolver function passed to ${symbol_dollar}Q

Expected resolverFn, got '{0}'

Description

Occurs when calling creating a promise using ${symbol_dollar}q as a constructor, without providing the required resolver function.

//bad
var promise = ${symbol_dollar}q().then(doSomething);

//good
var promise = ${symbol_dollar}q(function(resolve, reject) {
  waitForSomethingAsync.then(resolve);
}).then(doSomething);

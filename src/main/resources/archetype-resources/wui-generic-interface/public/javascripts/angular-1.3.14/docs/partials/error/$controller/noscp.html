Error: error:noscp
Missing ${symbol_dollar}scope object

Cannot export controller '{0}' as '{1}'! No ${symbol_dollar}scope object provided via `locals`.

Description

This error occurs when ${symbol_dollar}controller service is called in order to instantiate a new controller but no scope is provided via ${symbol_dollar}scope property of the locals map.

Example of incorrect usage that leads to this error:

${symbol_dollar}controller(MyController);
//or
${symbol_dollar}controller(MyController, {scope: newScope});

To fix the example above please provide a scope (using the ${symbol_dollar}scope property in the locals object) to the ${symbol_dollar}controller call:

${symbol_dollar}controller(MyController, {${symbol_dollar}scope: newScope});

Please consult the ${symbol_dollar}controller service api docs to learn more.

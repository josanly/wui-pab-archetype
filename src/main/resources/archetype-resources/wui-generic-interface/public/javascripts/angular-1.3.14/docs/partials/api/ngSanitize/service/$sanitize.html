${symbol_dollar}sanitize

  1. - service in module ngSanitize

The input is sanitized by parsing the HTML into tokens. All safe tokens (from a whitelist) are then serialized back to properly escaped html string. This means that no unsafe input can make it into the returned string, however, since our parser is more strict than a typical browser parser, it&${symbol_pound}39;s possible that some obscure input, which would be recognized as valid HTML by a browser, won&${symbol_pound}39;t make it through the sanitizer.
